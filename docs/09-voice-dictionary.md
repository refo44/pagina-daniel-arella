# Daniel Arella — Voice Dictionary v2.0

Léxico del sistema editorial: palabras permitidas y prohibidas para navegación, botones y mensajes.

No es un diccionario literario; es el **léxico donde vive la obra** en el sitio.  
Principios de voz y tono: ver `guia-voz-microcopy-ux.md`.

---

## 1. Principio rector

Si una palabra pertenece a marketing, ventas, startups, redes, growth o monetización, **no** pertenece a este sitio.

El sitio **acompaña una lectura**; no convierte, no optimiza, no retiene.

---

## 2. Verbos permitidos

Usar en botones, enlaces y mensajes:

| Verbo     | Uso |
|-----------|-----|
| Leer      | Acción principal |
| Abrir     | Libros, PDF, EPUB |
| Ver       | Listas, catálogos |
| Explorar  | Archivo, temas, periodos |
| Recibir   | Correspondencia, textos |
| Entrar    | Secciones |
| Volver    | Navegación |
| Continuar | Lectura |
| Adquirir  | Compra editorial |
| Descargar | Archivos |
| Contactar | Comunicación |
| Enviar    | Formularios |
| Guardar   | Preferencias |
| Cerrar    | Modales |
| Cambiar   | Idioma, vista |

**Nunca:** comprar, obtener, desbloquear, acceder, registrarse, suscribirse, activar, empezar.

---

## 3. Sustantivos permitidos

| Categoría     | Palabras |
|---------------|----------|
| Obra          | poema, libro, texto, ensayo, relato, cuaderno, fragmento |
| Tiempo        | periodo, etapa, año |
| Edición       | versión, edición, archivo |
| Publicación   | publicado, inédito, borrador |
| Comunicación  | carta, correspondencia |
| Personas      | lector, autor |
| Actividad     | lectura, escritura |
| Archivo       | biblioteca, archivo |

**Nunca:** producto, contenido, usuario, audiencia, lead, funnel, plataforma, dashboard.

---

## 4. Correo (correspondencia)

| Evitar        | Usar |
|---------------|------|
| Newsletter    | Correspondencia, Carta |
| Subscriber    | Lector |
| Signup        | Recibir |
| Mailing list  | Lista de lectores |

**Nunca (en UI):** suscriptor, audiencia, base de correos, leads, newsletter (salvo contexto legal si es imprescindible).

---

## 5. Estados editoriales

| Situación    | Palabra   |
|--------------|-----------|

**Nunca (texto visible):** draft, hidden, private, archived.

---

## 8. Mensajes del sistema

| Situación     | Texto |
|---------------|--------|
| Sin resultados | “No hay textos en esta sección.” / “No hay textos con estos filtros.” |
| Archivo vacío  | “Este archivo aún está en construcción.” |
| Error          | “No fue posible mostrar este texto.” |
| Eliminado      | “Este texto fue retirado.” |
| No disponible  | “Este contenido no está disponible.” |
| 404            | “Este texto no existe o fue retirado.” |

**Nunca:** Error 404, No data, Loading, Oops.

---

## 9. Títulos y encabezados

- **Secciones:** sustantivo simple (Poemas, Ensayos, Libros).
- **Listas:** plural.
- **Obras:** título real de la obra.

**Correcto:** Poemas, Ensayos, Libros, Archivo.  
**Incorrecto:** Mis poemas, Lo último de Daniel, Nuevos lanzamientos.

---

## 10. Regla final

Si suena a landing, app, startup, tienda o influencer, no puede estar aquí.

El sitio **acoge una obra**; no persuade ni empuja.
