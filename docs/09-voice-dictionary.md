# Daniel Arella — Voice Dictionary

**Léxico editorial y UX del sistema**  
**Versión 2.1**

Este diccionario define qué palabras existen dentro del territorio editorial de Daniel Arella y cuáles no pueden existir jamás. No es un diccionario literario. Es el léxico del espacio donde vive la obra.

Todas las interfaces, botones, menús, mensajes y formularios del sitio deben obedecer este documento.

La voz y el tono se definen en `08-guia-voz-microcopy-ux`.

---

## 1. Principio rector

Si una palabra pertenece al lenguaje de:

- marketing
- ventas
- growth
- plataformas
- startups
- redes sociales
- monetización

entonces no pertenece a este sitio.

Este sitio no es una aplicación. No es una tienda. No es una landing. El sitio acompaña una lectura. No persuade, no empuja, no captura.

---

## 2. Verbos permitidos

Usar exclusivamente estos verbos en botones, enlaces, microcopy y mensajes.

| Verbo | Uso |
|-------|-----|
| Leer | Acción principal de la plataforma |
| Abrir | Libros, PDFs, EPUB |
| Ver | Listas, catálogos |
| Explorar | Archivo, temas, periodos |
| Recibir | Correspondencia, textos |
| Entrar | Secciones |
| Volver | Navegación |
| Continuar | Lectura |
| Adquirir | Compra editorial |
| Descargar | Archivos |
| Contactar | Comunicación |
| Enviar | Formularios |
| Guardar | Preferencias |
| Cerrar | Modales |
| Cambiar | Idioma, vista |

**Nunca usar:** comprar, obtener, desbloquear, acceder, registrarse, suscribirse, activar, empezar, descubrir, probar, unirse, crear cuenta.

---

## 3. Sustantivos permitidos

| Categoría | Palabras |
|-----------|----------|
| Obra | poema, libro, texto, ensayo, relato, cuaderno, fragmento |
| Tiempo | periodo, etapa, año |
| Edición | versión, edición, archivo |
| Publicación | publicado, inédito, borrador |
| Comunicación | carta, correspondencia |
| Personas | lector, autor |
| Actividad | lectura, escritura |
| Archivo | biblioteca, archivo |

**Nunca usar:** producto, contenido, usuario, audiencia, cliente, lead, funnel, plataforma, dashboard, feed, perfil, tienda.

---

## 4. Correo y relación directa

El sistema no tiene newsletters. Tiene correspondencia.

| Evitar | Usar |
|--------|------|
| Newsletter | Correspondencia, Carta |
| Subscriber | Lector |
| Signup | Recibir |
| Mailing list | Lista de lectores |

**Nunca mostrar en UI:** suscriptor, base de correos, leads, audiencia, newsletter.

---

## 5. Estados editoriales

| Situación | Palabra |
|-----------|---------|
| No publicado | inédito |
| En proceso | borrador |
| Visible | publicado |
| Retirado | retirado |
| Archivado | archivo |

**Nunca mostrar:** draft, hidden, private, archived.

---

## 6. Navegación

| Función | Término |
|---------|---------|
| Home | Inicio |
| Archive | Archivo |
| Books | Libros |
| Poems | Poemas |
| Essays | Ensayos |
| Stories | Relatos |
| Workshops | Talleres |
| About | Sobre el autor |
| Contact | Contacto |
| Press | Prensa |
| Articles (si existen) | Artículos o Escritura |

**Nunca:** dashboard, feed, profile, store, shop, portal.

---

## 7. Botones estándar

| Función | Texto |
|---------|-------|
| Leer poema | Leer el poema |
| Leer libro | Abrir el libro |
| Email | Recibir nuevos textos |
| Compra | Adquirir el libro |
| Descarga | Descargar PDF o EPUB |
| Archivo | Explorar archivo |
| Idioma | Cambiar idioma |
| Envío | Enviar |
| Cierre | Cerrar |

**Nunca:** Ver más, Comprar ahora, Empieza aquí, Suscríbete, Descubre, Únete.

---

## 8. Mensajes del sistema

| Situación | Texto |
|-----------|-------|
| Sin resultados | “No hay textos en esta sección.” |
| Sin filtros | “No hay textos con estos filtros.” |
| Archivo vacío | “Este archivo aún está en construcción.” |
| Error | “No fue posible mostrar este texto.” |
| Eliminado | “Este texto fue retirado.” |
| No disponible | “Este contenido no está disponible.” |
| 404 | “Este texto no existe o fue retirado.” |

**Nunca:** Error 404, No data, Loading, Oops, Algo salió mal.

---

## 9. Títulos y encabezados

**Reglas:**

- Secciones usan sustantivos simples.
- Listas usan plural.
- Obras usan su título real.

**Correcto:** Poemas, Ensayos, Libros, Archivo

**Incorrecto:** Mis poemas, Lo último de Daniel, Nuevos lanzamientos, Lo que tienes que leer

---

## 10. Regla final

Si una palabra suena como si pudiera aparecer en:

- una landing
- una app
- una startup
- una tienda
- un perfil de influencer

entonces no puede estar aquí.

El sitio no vende. El sitio no empuja. El sitio acoge una obra.
